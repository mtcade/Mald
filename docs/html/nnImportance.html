One step interface to use maldImportance.importance without a model in hand by using maldImportance.superBasicNetworks.SimpleNN.

maldImportance.nnImportance.importances(X: ndarray | DataFrame, Xk: ndarray | DataFrame, y: ndarray | Series, local_grad_method: Literal['auto_diff', 'bandwidth'] = 'auto_diff', bandwidth: float | None = None, exponent: float = 2.0, drop_first: bool = True, save_root: str = '', save_name: str = '', epochs: int = 500, dense_activation: str = 'relu', first_layer_width: float | int = 0.25, layers: int = 2, layer_shrink_factor: float = 0.25, learning_rate: float = 0.01, verbose: int = 0) → ndarray¶
Parameters:

  • X (np.ndarray|pd.DataFrame) – Explanatory data

  • Xk (np.ndarray|pd.DataFrame) – Knockoff explanatory data

  • y (np.ndarray|pd.Series) – Outcome data

  • local_grad_method (Literal['auto_diff','bandwidth']) – Method of MALD. Defaults to ‘auto_diff’ for exact autodifferentiation. ‘bandwidth’ uses the bandwidth approximation when auto differentiation is not available.

  • bandwidth (float|None) – Width if local_grad_method = ‘bandwidth’

  • exponent (float) – Power to take of each MALD value. 1.0 and 2.0 both work reasonably well.

  • drop_first (bool) – How to handle one-hot-encoding categorical variables. If True the number of associated columns is the number of categories minus 1.

  • save_root (str) – Dir to make saves

  • save_name (str) – Name to use for saving checkpoints

  • epochs (int) – Epochs for .fit()

  • dense_activation (str) – Activation of internal layers, most likely ‘relu,’ ‘sigmoid’, ‘leaky_relu’, etc

  • first_layer_width (float|int) – Size of first layer after input. If an integer, it uses that value. If a float, it’s a multiplier of the input size.

  • layers (int) – Number of internal layers

  • layer_shrink_factor (float) – With multiple internal layers, the width of each is this multiplied by the previous layer’s width

  • learning_rate (float) – Learning rate for .fit(); common values are 0.01, 0.005, 0.001, 0.0005, 0.0001

  • verbose (int) – How much to print out, for mostly for debugging.

Returns:

Array of importances, with length equal to twice the width of X

Return type:

np.ndarray

Creates a SuperBasicNetwork and uses it for MALD Importance. See <no title> and <no title>
